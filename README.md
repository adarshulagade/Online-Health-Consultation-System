# Online Health Consultation System

## 📌 Description
A web-based system that allows patients to consult doctors online, book appointments, and receive prescriptions digitally.

## 💻 Technologies Used
- PHP
- MySQL
- HTML, CSS
- XAMPP

## ⚙️ Features
- Patient Registration & Login
- Book Appointment
- Doctor Dashboard
- Approve/Reject Appointments
- Add Prescription
- Admin Panel (Manage patients, doctors, appointments)

## 🚀 How to Run
1. Download or clone repository
2. Place in `xampp/htdocs`
3. Start Apache & MySQL in XAMPP
4. Import database (`health_consultation_db`)
5. Open browser:  
   `http://localhost/online_consultation`
